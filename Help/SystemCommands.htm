System Commands

)

Restore the last statement to the input area.

)Fns

Lists the method definitions.

)Vars

    Lists the variables currently defined.

)Clear

          Removes all functions and variables.

)Erase Names

          Removes one or more specified names.

)Edit {Name}

Edits the elements in the specified name, or if no name is provided,

commands entered since the last )Load are available.  Please note;

a vector will be returned if you provide a different number of values. 

 

)Lib

    Lists the workspaces in the library.

)Load wsName

Restore the values and functions from a previous )Save.

)Copy wsName {Names}

Merges the values and functions from a previous )Save.

)Save {wsName}

Preserves the values and function definitions from this session.

)Drop wsName

Removes a saved workspace.

 

)Compile

    Launch the  VB.Net compiler.

)Version

    Displays the current version number, and provides a link

to determine if you have the latest version.
