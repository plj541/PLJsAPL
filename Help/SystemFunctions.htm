ShellAPL Provides

Commonly Available System Functions

 

          ⎕Def aMethod

Returns the definition of aMethod.

aDef ⎕Def aMethod

Makes aDef the definition of aMethod.

          ⎕EX aListOfNames

Erases the names in the list.

          ⎕NL 2 or 3

Returns a list of values or methods.

          ⎕Fi

Formatted Input, i.e. turn characters into numbers.  Invalid input will be represented as 0.

          ⎕Vi

Verify Input will have 1 where the input is valid and 0 where it is not.

 

Functions to Read and Write Files

 

          ⎕Lines aFile

Reads the named file.

myData ⎕Lines aFile

Writes myData to the named aFile.

This is will be a UTF-8 file.

          ⎕Bytes aFile

Reads aFile and translates as specified by ⎕BytesAV.

myData ⎕Bytes aName

Writes translated data to the named file.  Only characters in ⎕BytesAV are acceptable.

This is will be an ANSI file.

⎕BytesAV←⎕AvAPLse

Sets translate for  Bytes.

          ⎕AvEbcdic and ⎕AvAPL2pc

are also provided.  Any character vector will do.

          ⎕Files aPath

Returns a list of file names.

          ⎕Paths aPath

Returns a list of folders.

          ⎕Path

Returns the current folder.

          ⎕Path←aNewFolder

Changes the current folder.  Paths to several standard folders are available.

⎕Host.Desktop

⎕Host.MyDocuments

⎕Host.ProgramFiles

⎕Host.Temp

 

Functions to Link to Compiled Workspaces and Classes

 

          ⎕Lib ""

Returns a list of compiled workspaces.

          ⎕Lib aDll

Returns a list of compiled Classes in aDll.

          ⎕Fns aWsid

Returns a list of method headers in aWsid.

aDll ⎕Fns aClass

Returns a list of method headers in aClass.

aLink←⎕LinkTo aWsid

Creates an instance of aWsid, and aLink provides access to methods in the instance.
