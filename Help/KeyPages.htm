The ShellAPL Keyboard

 

There are four levels to the default keyboard.  These are normal, shifted, Alt depressed and Alt plus Shift depressed.  I’ve displayed the default North American keyboard here for reference.  Your normal keyboard is unaffected.  Only the Alt and Shifted Alt are modified.  The character mapping follows the behavior of your keyboard, not this layout.

The configuration file ships as:

Program Files\PLJ\APL.PLJ\KeyPagesX.apl

Please rename it to KeyPages.apl if you want to use your own configuration. 

 

If you edit this file, both EditAPL and ShellAPL will behave as you’ve specified.
